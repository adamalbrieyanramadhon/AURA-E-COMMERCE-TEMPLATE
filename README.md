# AURA PRO — High-Converting Minimalist E-Commerce Template

![AURA PRO Preview](https://images.unsplash.com/photo-1523275335684-37898b6baf30?auto=format&fit=crop&w=1200&q=80)

AURA PRO adalah template *landing page* e-commerce premium dengan desain modern ultra-minimalis yang dirancang khusus untuk meningkatkan konversi penjualan *luxury brands*. Dibangun menggunakan teknologi ringan tanpa *framework* berat, menjamin kecepatan *loading* maksimal dan performa SEO yang superior.

---

## ✨ Fitur Unggulan (Conversion Boosters)

Template ini dilengkapi dengan berbagai fitur psikologi penjualan dan interaktivitas modern:

* **⏳ Real-Time Countdown Timer:** Bar promo melayang di bagian atas untuk memicu urgensi (*scarcity & urgency*).
* **🎨 Dynamic Variant Selector:** Pengubah warna produk interaktif yang langsung mengganti gambar utama secara instan tanpa memuat ulang halaman.
* **🛒 Slide-Out Cart Canvas:** Keranjang belanja interaktif gaya Shopify yang muncul dengan animasi halus dari sisi kanan.
* **🔔 Live Sales Notification Popup:** Notifikasi transaksi otomatis secara berkala di pojok kiri bawah untuk membangun bukti sosial (*social proof/FOMO*).
* **📱 Sticky Mobile Checkout Bar:** Tombol beli statis di bagian bawah khusus untuk pengguna *smartphone*, memudahkan proses *checkout*.
* **💬 Interactive FAQ Accordion:** Seksi tanya-jawab yang rapi dengan fungsi *toggle* buka-tutup yang interaktif.
* **🎬 Video Presentation Modal:** Fitur *popup* video untuk menyematkan *brand film* atau demonstrasi produk.

## 💻 Cara Penggunaan & Instalasi

### 1. Jalankan di Lokal (Komputer Anda)
1. Unduh atau salin file `index.html`.
2. Klik ganda file `index.html` untuk membukanya langsung di browser pilihan Anda (Chrome, Safari, Edge).

### 2. Aktifkan Demo Live di GitHub Pages
1. Masuk ke repositori GitHub tempat Anda menyimpan kode ini.
2. Pergi ke tab **Settings** > **Pages** (di menu sebelah kiri).
3. Pada bagian **Build and deployment**, ubah Source menjadi `Deploy from a branch`.
4. Pilih branch `main` (atau `master`) dan folder `/ (root)`, lalu klik **Save**.
5. Tunggu 1–2 menit, GitHub akan memberikan tautan *live website* demo Anda secara otomatis.

---

## 🛠️ Panduan Kustomisasi

Semua konfigurasi warna dan data produk diatur langsung di dalam satu file `index.html`:

* **Mengganti Warna Tema Utama:** Cari blok `tailwind.config` di bagian `<head>` dan ubah kode hex warna pada variabel `auraAccent` atau `auraBg`.
* **Mengganti Data Pembeli (Sales Popup):** Cari array `const buyers` di bagian paling bawah kode JavaScript untuk mengubah nama dan kota yang muncul pada notifikasi penjualan otomatis.
