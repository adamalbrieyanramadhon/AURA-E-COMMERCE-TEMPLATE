# AURA PRO — High-Converting Minimalist E-Commerce Template

![AURA PRO Preview](https://images.unsplash.com/photo-1523275335684-37898b6baf30?auto=format&fit=crop&w=1200&q=80)

AURA PRO is a premium, high-converting e-commerce landing page template featuring an ultra-minimalist design tailored for modern luxury brands. Built with a lightweight, clean-code approach without heavy frameworks, it ensures maximum loading speed and superior SEO performance out of the box.

---

## ✨ Key Features (Conversion Boosters)

This template is packed with psychological triggers and modern interactive elements designed to maximize sales:

* **⏳ Real-Time Countdown Timer:** A top announcement bar that drives urgency and scarcity.
* **🎨 Dynamic Variant Selector:** An interactive color picker that changes the main product image instantly without page reloads.
* **🛒 Slide-Out Cart Canvas:** A seamless Shopify-style interactive shopping cart that slides smoothly from the right viewport.
* **🔔 Live Sales Notification Popup:** Automated purchase notifications at the bottom-left to build immediate social proof and FOMO.
* **📱 Sticky Mobile Checkout Bar:** A floating call-to-action button pinned to the bottom of mobile screens to streamline mobile checkouts.
* **💬 Interactive FAQ Accordion:** A clean Q&A section with smooth toggle accordion functionality.
* **🎬 Video Presentation Modal:** An integrated popup modal for embedding brand films or product demonstrations.

## 💻 Installation & Usage

### 1. Running Locally
1. Download or clone the `index.html` file to your local machine.
2. Double-click the `index.html` file to open it directly in any modern browser (Chrome, Safari, Edge, Firefox).

### 2. Deploying a Live Demo via GitHub Pages
1. Navigate to your GitHub repository.
2. Go to the **Settings** tab > click **Pages** on the left-side menu.
3. Under **Build and deployment**, ensure the source is set to `Deploy from a branch`.
4. Select the `main` (or `master`) branch and `/ (root)` folder, then click **Save**.
5. Wait 1–2 minutes, and GitHub will automatically generate your live website demo link.

---

## 🛠️ Customization Guide

All configurations, styles, and product data are centralized inside the single `index.html` file:

* **Changing Theme Colors:** Locate the `tailwind.config` block inside the `<head>` section and update the hex color codes in variables like `auraAccent` or `auraBg`.
* **Updating Mock Buyer Data (Sales Popup):** Locate the `const buyers` array at the bottom of the JavaScript code to modify the names and cities displayed in the automated notification loop.
